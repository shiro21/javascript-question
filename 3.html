변수에 대하여( var, let, const )

변수란: 변수 이름에 값을 저장하는 것.

변수의 특징: 선언, 할당, 범위가 있습니다.

var의 특징: 재선언O, 재할당O, 범위: function입니다.( 함수 레벨 스코프 )

let의 특징: 재선언X, 재할당O, 범위: block입니다.( 블록 레벨 스코프 )

const의 특징: 재선언X, 재할당X, 범위: block입니다.( 블록 레벨 스코프 )

재선언예시
var name = "jun"
var name = "new"

재할당예시
let name = "jun"
name = "new"

재할당예시2
const people = { name: "Jun" }
people.name = "Park";
const로 object를 만들었을 때, 오브젝트 내부 값을 변경해도 에러가 나지 않습니다.( 값을 변경했을 뿐이고, 재할당한게 아닙니다. )
바꿀 수 없는 object를 만드려면 Object.freeze(people); 로 수정불가능하게 만들 수 있습니다.

범위예시
function func() { var name = "Jun" } var는 function내부에서 만들었을 경우 function내부에서만 사용할 수 있습니다.

범위예시2
if() { let name = "Jun"}
for (let i = 0; i < 5; i++) {}; let과 const는 var보다 범위가 더 좁습니다. {} 중괄호 내부에서 만들었을 때, 중괄호 내부에서만 사용할 수 있습니다.
